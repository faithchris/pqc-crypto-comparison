You opened the HTML file directly in the browser. The buttons cannot talk to Python that way.

Run the Flask app in a terminal, then use the URL it prints (usually http://127.0.0.1:5000).

  1. cd to your project folder (where encryptionCompare_v1.py lives).
  2. source .venv/bin/activate — or create a venv and pip install flask cryptography dilithium-py kyber-py.
  3. python encryptionCompare_v1.py
  4. Open http://127.0.0.1:5000 in the browser — not the templates/index.html file.
